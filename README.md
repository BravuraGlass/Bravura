# BRAVURA CRM

## Requirements

* Ruby version

2.3.4

* System dependencies

Run `bundle install`

## Configuration

1. Rename the railsenvironment-env.template.yml to the accordingly, for example `development-env.yml`, `production-env.yml`

2. Configure database.yml
3. run `rake db:migrate`
4. `rake bower:install`
